# SiteCacheEraser

## 清除Redis缓存，通过配置删除redis缓存

1. 选择配置的环境，选择配置的模块，选择要清楚的key，已进行清除
2. 根据Key搜索Redis的值并清除
 ![图片](https://dn-coding-net-production-pp.qbox.me/d48557b0-1fe0-479b-b7ab-e29f6337944b.png) 
3. 点击重启IIS将会执行配置的脚本【重启iis】
